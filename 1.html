HTML Tags

<tag's name>  ~ </tag's name> 

[Must use Tags]
<!doctype html>  Describes doc type
<html>  The root element
<head>  The document head
<meta charset="utf-8">  Read utf-8(Foreign language)
<body>  Contents

[Basic Tags]
<title>  The webpage's title
<strong>  Font strong
<u>  underline
<h1>  A section heading with a new line(h1~h6)
<br>  new line, don't use close tag
<p>  paragraph
<p style="margin-top:40px;">  add paragraph's style
<img src="image address" width="400">  add an images(image address is file's name)
<a>  A link
href="link address"
target="_blank"
title="tool tip"
ex) <a href="https://opentutorials.org/course/1" target="_blank" title="생활코딩 home"> 

[parents and child tags-always use together]
<li>  list (child)
